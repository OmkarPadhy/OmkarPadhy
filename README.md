### 📡 FPGA-Based QPSK Communication System with AWGN Channel
> Complete physical-layer digital communication system implemented on hardware

Designed and implemented a full QPSK transceiver on **Xilinx Basys-3 FPGA** in Verilog HDL. Built a PRBS generator, QPSK modulator/demodulator, and a hardware-based AWGN channel model. System performance validated using BER and EVM metrics with MATLAB-based analysis.

**Tech:** `Verilog HDL` `Xilinx Vivado` `MATLAB` `Basys-3 FPGA` `QPSK` `AWGN`
